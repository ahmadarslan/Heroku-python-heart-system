Titanic Passenger Survival Prediction

Project Aim

The main purpose of this project is to predict whether a passenger onboard on Titanic Ship, survived or not?


Input and output

Input: Passenger

Output: Prediction (Survived / Not Survived)


Representation of Input (Passenger)

         Input (Passenger) is Represented as 4 Input Attributes

         Input Attributes 01– PClass (Passenger Class)

                Possible Value 01 – First Class

                Possible Value 02 – Second Class

                Possible Value 03 – Third Class


         Input Attributes 02– Gender

                Possible Value 01 – Male

                Possible Value 02 – Female


         Input Attributes 03– Siblings

                Possible Value 01 – One

                Possible Value 02 – Two

                Possible Value 03 – Three


         Input Attributes 04– Embarked

                Possible Value 01 – Cherbourg

                Possible Value 02 – Queenstown

                Possible Value 02 – Southamption


Representation of Output

         Output is Represented as a single Output Attributes

         Output Attribute – Survived

                Possible Value 01 – Yes

                Possible Value 02 – No


Example - Input and Output

Input (Passenger):

         PClass:

                First Class

         Gender:

                Male

         Siblings:

                Two

         Embarked:

                Queenstown

Output: Survived
