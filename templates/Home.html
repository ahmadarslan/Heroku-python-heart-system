Heart Disease Prediction System

Project Aim

The main purpose of this project is to predict whether a patient with heart disease, survived or not?


Input and output

Input: Patient

Output: Prediction (Survived / Not Survived)


Representation of Input (Patient)

  Input (Patient) is Represented as 4 Input Attributes

  Input Attributes 01– Age (Patient Age)


  Input Attributes 02– Gender

    Possible Value 01 – Male

    Possible Value 02 – Female


  Input Attributes 03– Blood Pressure (BP Values)


  Input Attributes 04– Cholesterol Value


Representation of Output

   Output is Represented as a single Output Attributes

    Output Attribute – Survived

    Possible Value 01 – Yes

    Possible Value 02 – No


Example - Input and Output

Input (Patient):

  Age:

    24

  Gender:

    Male

  BP:

    187

  Cholesterol:

    233

Output: Survived
